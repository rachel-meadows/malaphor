What is a 'malaphor'?

A malaphor is a blend of common sayings or idioms. For example, "We'll burn that bridge when we come to it" is a cross between "burn your bridges" and "we'll cross that bridge when we come to it".

How does this site work?

This site runs off a Python program that takes Wictionary's list of idioms and proverbs, and uses Python and SQLite to merge idioms together.

There are a series of algorithms the program runs through, with preference given to pairs that are semantically similar or contain similar text (or both!).
